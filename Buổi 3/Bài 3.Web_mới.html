Hiểu hơn về thiết kế Web

Thiết kế web là việc tạo ra một trang web cho cá nhân, tổ chức, công ty hay doanh nghiệp. Thiết kế web gồm thiết kế web tĩnh và động:
Thiết kế web tĩnh: Là việc dùng các đoạn mã HTML, video, hình ảnh, Flash, Javascript, CSS để tạo giao diện web. Tên tập tin được lưu sẽ có phần mở rộng là .html hoặc .htm. Thiết kế web tĩnh không có hệ CSDL như MySQL và MSSQL.
Thiết kế web động: Là web có hệ thống CSDL cung cấp thông tin cho website. Ưu điểm của web động là khả năng tương tác trên hệ thống web, quản lý dữ liệu web tốt, dễ dàng cập nhật nội dung, thêm tính năng quản lý đơn giản và thân thiện với người sử dụng.
